# 🏢 东莞市既有住宅增设电梯投票表决计算器

一个基于 HTML/CSS/JavaScript 开发的轻量级网页工具，专为**东莞市**既有住宅增设电梯的业主委员会或居民设计。该工具严格依据《东莞市既有住宅增设电梯管理办法（修订）》的表决规则，帮助用户快速计算投票结果，判断是否满足增设电梯的法定条件。

> **当前状态**：本办法有效期至 2026 年 5 月 2 日（请根据实际政策更新情况使用）。

---

## 📜 法律依据

本计算器严格遵循 **《东莞市既有住宅增设电梯管理办法（修订）》第六条** 的规定：

> **“由本单元或者本幢房屋专有部分面积占比三分之二以上的业主且人数占比三分之二以上的业主参与表决，经参与表决专有部分面积四分之三以上的业主且参与表决人数四分之三以上的业主同意。”**

> *注：本代码中暂按“户数”等同于“专有部分面积”进行简易计算（即每户面积视为均等），符合一般老旧小区的实际情况。若需精确计算建筑面积，请自行扩展输入项。*

---

## ✨ 核心功能

-   **双指标计算**：同时计算 **户数** 和 **业主人数** 两个维度的投票比例。
-   **四重条件判定**：
    1.  参与表决户数是否 $\ge 2/3$。
    2.  参与表决人数是否 $\ge 2/3$。
    3.  同意户数占参与表决户数是否 $\ge 3/4$。
    4.  同意人数占参与表决人数是否 $\ge 3/4$。
-   **可视化反馈**：
    -   🟢 **绿色**：通过（显示“🎉 投票通过”）。
    -   🔴 **红色**：未通过（明确提示是参与率不足还是同意率不足）。
-   **响应式设计**：适配桌面端和移动端查看。

## 🛠️ 如何使用

你可以直接在浏览器中打开 `index.html` 文件使用，或者将其部署为静态网站。

### 1. 数据录入
在表格中输入每户的基本信息：
-   **房号**：系统自动生成（可手动修改）。
-   **业主人数**：输入该户的产权人数量（默认为 1）。
-   **是否同意**：勾选同意安装的业主。

### 2. 执行计算
点击底部的 **“计算投票结果”** 按钮。

### 3. 查看结果
系统将自动展示：
-   **统计卡片**：总户数、总人数、参与率、同意率。
-   **状态提示**：顶部的大框会显示最终的通过状态。
-   **详细判定**：下方列出具体的 4 个判断条件是否满足。
